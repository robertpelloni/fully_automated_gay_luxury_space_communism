# AI Hustle Machine — Fully Automated Luxury Protocol

A comprehensive, self-evolving, federated monorepo for orchestrating AI-driven revenue streams.

## 🚀 Unified Monorepo Architecture
The project is organized into highly decoupled modules that communicate via the `hustle://` protocol:

### Core Infrastructure
- **Orchestrator**: The central hub for task scheduling, mesh networking, and capital governance.
- **Mesh Swarm**: Federated memory and status aggregation across distributed agent nodes.
- **Wealth Preservation**: Automated ROI auditing and capital defense logic.

### Hustle Modules
- **Research**: Sentiment extraction and ticker discovery.
- **Curation**: Automated news processing and blurb generation.
- **Social**: Multi-platform authenticated posting.
- **Trading**: Confluence-based execution with real-world price data.

## 🧠 Engineering Workflows
Standardized gstack engineering skills are integrated natively into `.agent/workflows/` for use with Antigravity and Gemini CLI:
- `/office-hours`: Brainstorm and validate product ideas.
- `/plan-ceo-review`: Strategic scope and user-centric review.
- `/plan-eng-review`: Architecture and edge-case validation.
- `/ship`: Automated release and verification.
